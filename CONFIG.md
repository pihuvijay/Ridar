# Configuration Files

## Frontend
- **`app.json`** - Expo app configuration (metadata, icons, splash screen, platform settings)
- **`tsconfig.json`** - TypeScript configuration for Expo React Native frontend
- **`package.json`** - Frontend dependencies and scripts (Expo, React Native)
- **`babel.config.js`** - Babel transpiler configuration for React Native

## Backend  
- **`tsconfig.json`** - TypeScript configuration for Node.js Express backend
- **`package.json`** - Backend dependencies and scripts (Express, TypeScript)
- **`.env`** - Environment variables (Supabase keys, database URLs)
- **`.env.example`** - Template for environment variables

## Root
- **`package.json`** - Workspace configuration for monorepo setup
